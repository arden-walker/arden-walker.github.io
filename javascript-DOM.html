What is JavaScript and how does it fit with HTML and CSS:

JavaScript is the language of the web. JavaScript is what is used to communicate with the browser, giving you all the cool stuff you can do on all the big websites. HTML is the structure of your webpage. If we think of our websites as houses than HTML is your walls, your doors and your ceilings. CSS is the colours, the things that make your house pretty and JavaScript is what gives your house (webpage) it's functions like opening and closing doors, windows, cupboards light switches and taps etc. Combined together, they are a full house.

What is the DOM:

-The DOM = "THE DOCUMENT OBJECT MODEL" it is what we use to inspect our house (webpage). It is used so that programs can change the structure, the style and the content by representing the documents as nodes and objects. It is also used so that other programming languages can interact with the page.

What are NODES in the DOM:

Nodes are individual elements, attributes, and text we use in HTML code. They are used to identify and process data and can perform multiple functions depending on the application.

Explain the difference between accessing data from arrays and objects:

-Arrays are used when the elements are numbers. Objects are used when the elements are strings. The data inside an array is known as Elements. The data inside objects are known as Properties which consists of a key and a value.

Explain what functions are and why they are helpful:

-There are two parts to a function. There's the declaring part and then there's the calling of a function. We call a function so that we don't have to keep repeating ourselves and creating an absolute mess in our code. We declare a function by filling braces "{ }" with what our function is going to do. e.g. Function Declare-Function() { declare function here }
